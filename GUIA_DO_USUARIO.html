01

Instalação

Instalação simples e rápida direto da Chrome Web Store oficial.

Instalar Agora
1
Acesse a Chrome Web Store

Clique no botão "Instalar Agora" ou acesse o link da extensão no seu navegador Chrome.

2
Clique em "Usar no Chrome"

Na página da extensão, clique no botão azul "Usar no Chrome" (ou "Add to Chrome").

3
Confirme a Instalação

Um popup pedirá permissão. Clique em "Adicionar extensão" para confirmar.

4
Pronto!

A extensão será instalada. Você verá o ícone do AuvoTools na barra de extensões.

Fixar a Extensão (Recomendado)

Clique no ícone de quebra-cabeça no Chrome, encontre "AuvoTools" e clique no alfinete para fixar na barra.

Menu de Navegação

Acesse qualquer página do Auvo com apenas 2 cliques através do menu dropdown.

Como Usar
  1. Acesse qualquer página do Auvo (app.auvo.com.br ou cs.auvo.com.br)
  2. Procure o botão "Acessar Conta" no topo
  3. Clique na setinha ao lado do botão
  4. Escolha a página desejada no menu dropdown

Dica: Navegação com Código

Selecione um cliente/colaborador/tarefa no Select2 do Auvo, depois use o menu. A extensão pegará o código automaticamente e abrirá a página já filtrada!

Aba Servidor

Teste de conexão com os servidores do Auvo Brasil para diagnóstico rápido.

Quando Usar Esta Função?
  • Quando o sistema Auvo estiver lento ou com erros
  • Quando uma página não carregar corretamente
  • Para verificar se o problema é no servidor ou na sua conexão
  • Quando um cliente reportar lentidão e você precisar identificar qual servidor ele usa
Como Testar os Servidores
  1. Abra o menu (setinha ao lado de "Acessar Conta")
  2. Clique na aba "Servidor"
  3. Você verá a lista dos 4 servidores disponíveis
  4. Clique em um servidor para testar a conexão
Servidor URL Observação
App1app1.auvo.com.brServidor Padrão
App2app2.auvo.com.brServidor secundário
App3app3.auvo.com.brServidor secundário
App4app4.auvo.com.brServidor secundário
Dica

O App1 é o servidor padrão onde nós aqui da Auvo estamos logados. Caso o cliente entre em contato relatando lentidão, teste os outros servidores (App2, App3, App4) para verificar se o problema é geral ou específico de um servidor.

05

Assistente IA

Chatbot treinado com os manuais da Auvo para tirar dúvidas rapidamente.

Como Acessar o Chat
  1. Acesse qualquer página do Auvo
  2. Procure o balão roxo no canto inferior direito
  3. Clique no balão para abrir o chat
  4. Digite sua pergunta e pressione Enter
Exemplos de Perguntas

O assistente buscará no Guia de Ativação Omie e explicará o passo a passo para configurar a integração ERP.

O assistente explicará o que é o Código Nacional de Atividades Econômicas e como encontrar o CNAE correto.

O assistente consultará o Manual de Nota Fiscal e listará os passos para emissão de NFS-e.
Limitações do Chat
  • O assistente só responde sobre assuntos dos manuais da Auvo
  • Ele não tem acesso a dados específicos de clientes ou contas
  • Para dúvidas muito específicas, consulte um supervisor
  • Se a resposta não for satisfatória, tente reformular a pergunta

Análise de Erros NF

Entenda erros de Nota Fiscal em linguagem simples com ajuda da IA.

Como Funciona
  1. Acesse a página de Nota Fiscal no Auvo
  2. Quando um erro aparecer (texto vermelho), procure o botão roxo "ANALISAR ERRO (IA)"
  3. Clique no botão
  4. Um popup aparecerá com a explicação do erro em linguagem simples
  5. O popup também sugere possíveis soluções
Tipo de Erro Exemplos
Rejeição SEFAZRejeição 204, Rejeição 539, Rejeição 778
Erro de SchemaErro de validação XML, campo obrigatório
Erro de NCMNCM inválido, NCM não encontrado
Erro de CFOPCFOP incompatível, CFOP inválido
Erro de CertificadoCertificado expirado, certificado inválido
Dica

A IA analisa o texto do erro e busca informações nos manuais para dar uma explicação clara e sugerir como resolver. Use essa funcionalidade para entender erros técnicos sem precisar pesquisar manualmente!

08

Dicas e Boas Práticas

Atalhos Favoritos

Configure apenas as páginas que você mais usa no menu para acesso mais rápido.

Perguntas Claras

Ao usar o chat, faça perguntas diretas e específicas para melhores respostas.

Prints Limpos

Ative o "Modo Print Limpo" antes de tirar screenshots para clientes.

Modo Anônimo

A extensão também funciona no modo anônimo (ative nas configurações do Chrome).

Dicas para o Chat IA
  • Seja específico: "Como configurar Omie" é melhor que "integração"
  • Use termos técnicos: CNAE, NCM, CFOP são reconhecidos
  • Uma pergunta por vez: Evite perguntas muito longas ou múltiplas
  • Reformule se necessário: Se não encontrar, tente sinais diferentes
